06. COLOR SYSTEM SUMMARY

Total Palette: 21 defined colors

• 2 base palettes (6 colors): CARBON (dark) + CAMO (light)

• 6 agent signature colors: Analyst, PM, Architect, DEV, SM, UX Designer

• 5 semantic status colors: Success, Error, Warning, Info, Neutral

• 4 highlight accent colors: Coral, Sky Blue, Mint, Orange


Design Philosophy:

Dual-layer strategy where neutral bases provide professional backdrop, while agent colors create identity and semantic colors add meaning without visual noise.


Accessibility:

• All status colors meet WCAG 2.1 Level AA contrast requirements

• Status never relies on color alone (always paired with icons/labels)

• Colorblind-safe palette tested with deuteranopia, protanopia, tritanopia
