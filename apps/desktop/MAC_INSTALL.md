# Chiron For macOS

## Install

1. Download the `Chiron-<version>-arm64.dmg` file.
2. Open the `.dmg`.
3. Drag `Chiron.app` into `Applications`.
4. Open `Applications` and launch `Chiron`.

## First Launch

Because this build is not notarized yet, macOS may warn that the app is from an unidentified developer.

If that happens:

1. In Finder, open `Applications`.
2. Right-click `Chiron.app` and choose `Open`.
3. Click `Open` again in the system prompt.

If macOS still blocks the app:

1. Open `System Settings`.
2. Go to `Privacy & Security`.
3. Find the message about `Chiron` near the bottom.
4. Click `Open Anyway`.

## Notes

- This build is for Apple Silicon Macs (`arm64`).
- Chiron stores its local app data in the standard macOS user application data location.
